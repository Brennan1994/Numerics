Detailed Description

Contains methods for numerical differentiation.

Authors: Haden Smith, USACE Risk Management Center, cole..nosp@m.h.sm.nosp@m.ith@u.nosp@m.sace.nosp@m..army.nosp@m..mil

Description: These methods estimate the derivative of a function using values and other knowledge about the function.
