static double Numerics.Mathematics.SpecialFunctions.Evaluate.Polynomial ( double[] coefficients,
double x )
static

Evaluates a double precision polynomial.

For sanity's sake, the value of N indicates the NUMBER of coefficients, or more precisely, the ORDER of the polynomial, rather than the DEGREE Of the polynomial. The two quantities differ by 1, but cause a great deal of confusion.

For example, a polynomial of order 4 in this function would be: coefficients[3]*x^3 + coefficients[2]*x^2 + coefficients[1]*x + coefficients[0]

References:

  • Based on a function contained in algorithm AS241, Applied Statistics, 1988, Vol. 37, No. 3.
Parameters
coefficientsThe coefficients of the polynomial. Item[0] is the constant term.
xThe point at which the polynomial is to be evaluated.
Returns
The polynomial with the given coefficients evaluated at x

static

Evaluates a double precision polynomial. Coefficients are in reverse order.

For example, a polynomial of order 4 in this function would be: coefficients[0]*x^3 + coefficients[1]*x^2 + coefficients[2]*x + coefficients[3]

Parameters
coefficientsThe coefficients of the polynomial. The last element in the list is the constant term.
xThe point at which the polynomial is to be evaluated.
nOptional parameter to redefine the order of the polynomial to be n+1
Returns
The polynomial with the given coefficients in reverse order evaluated at x

Evaluates a double precision polynomial. Coefficients are in reverse order, and coefficient(N) = 1.0.

For example, a polynomial of order 4 in this function would be: x^3 + coefficients[0]*x^2 + coefficients[1]*x + coefficients[2]

Parameters
coefficientsThe coefficients of the polynomial. The last element in the list is the constant term, and the first element is the coefficient for the second term, the coefficient for the first term is always 1
xThe point at which the polynomial is to be evaluated
Returns
The polynomial with the given coefficients in reverse order, with 1 as the coefficient for the first term, evaluated at x
