Detailed Description

A class for solving a set of linear equations using LU Decomposition.

Authors:

  • Haden Smith, USACE Risk Management Center, cole..nosp@m.h.sm.nosp@m.ith@u.nosp@m.sace.nosp@m..army.nosp@m..mil
  • Tiki Gonzalez, USACE Risk Management Center, julia.nosp@m.n.t..nosp@m.gonza.nosp@m.lez@.nosp@m.usace.nosp@m..arm.nosp@m.y.mil

Description:

LU decomposition is a useful method utilizing Gaussian Elimination to decompose a matrix into a lower and upper triangular matrix.

References:

  • "Numerical Recipes: The art of Scientific Computing, Third Edition." Press et al. 2017.
  • "Applied Numerical Methods with MATLAB for Engineers and Scientists, Third Edition.", Steven C. Chapra, McGraw-Hill, 2012.
