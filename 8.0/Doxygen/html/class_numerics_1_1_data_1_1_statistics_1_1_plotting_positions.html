Detailed Description

Contains plotting position formulas.

Authors: Haden Smith, USACE Risk Management Center, cole..nosp@m.h.sm.nosp@m.ith@u.nosp@m.sace.nosp@m..army.nosp@m..mil

Description: The choice of quantiles from a theoretical distribution can depend upon context and purpose. Several different formulas have been used or proposed as affine symmetrical plotting positions. Such formulas have the form (k − a) / (n + 1 − 2a) for some value of a in the range from 0 to 1, which gives a range between k / (n + 1) and (k − 1) / (n - 1).
