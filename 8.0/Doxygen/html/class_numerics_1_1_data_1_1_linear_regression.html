Detailed Description

A class for performing linear regression.

Authors: Haden Smith, USACE Risk Management Center, cole..nosp@m.h.sm.nosp@m.ith@u.nosp@m.sace.nosp@m..army.nosp@m..mil

Description: Linear regression works to estimate the linear relationship between a single, scalar response (dependent) variable and one or more explanatory (independent) variables. This implementation estimates Y = α + βX + e, where e ~ N(0,σ), and fits the linear regression model using singular value decomposition (SVD).
