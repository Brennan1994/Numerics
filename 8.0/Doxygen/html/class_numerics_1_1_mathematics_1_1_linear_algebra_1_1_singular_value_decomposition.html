Detailed Description

A class for solving a set of linear equations using Singular Value Decomposition.

Authors: list type = "bullet" > Haden Smith, USACE Risk Management Center, cole..nosp@m.h.sm.nosp@m.ith@u.nosp@m.sace.nosp@m..army.nosp@m..mil Tiki Gonzalez, USACE Risk Management Center, julia.nosp@m.n.t..nosp@m.gonza.nosp@m.lez@.nosp@m.usace.nosp@m..arm.nosp@m.y.mil

Description:

Singular value decomposition (SVD) is one of the most useful matrix decomposition methods. Used in:

  • Reducing dimension of large data sets (PCA)
  • Separating "signal" from "noise" in real-world data
  • Identifying line / low-dim plane of best-fit to high dimensional data
  • Solving/interpreting linear systems of equations
