Numerics provides a comprehensive set of methods and algorithms for numerical computations and statistical analysis, serving as a cornerstone of the RMC software suite. Detailed Description

A class for a simple linear function, with a single predictor and a slope and intercept coefficient, and normally distributed noise. Y = α + βX + ϵ, where ϵ ~ N(0,σ)

Authors: Haden Smith, USACE Risk Management Center, cole..nosp@m.h.sm.nosp@m.ith@u.nosp@m.sace.nosp@m..army.nosp@m..mil
