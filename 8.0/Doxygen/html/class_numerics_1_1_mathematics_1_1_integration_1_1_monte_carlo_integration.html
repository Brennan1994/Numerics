Detailed Description

A class for Monte Carlo integration for multidimensional integration.

Authors: Haden Smith, USACE Risk Management Center, cole..nosp@m.h.sm.nosp@m.ith@u.nosp@m.sace.nosp@m..army.nosp@m..mil

Description: This method numerically computes a definite integral by randomly choosing points at which the integrand is evaluated (rather than evaluating the integral at regular grid like other methods).
