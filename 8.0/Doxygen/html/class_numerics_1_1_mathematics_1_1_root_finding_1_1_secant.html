Detailed Description

The secant root-finding algorithm.

Authors:

  • Haden Smith, USACE Risk Management Center, cole..nosp@m.h.sm.nosp@m.ith@u.nosp@m.sace.nosp@m..army.nosp@m..mil
  • Tiki Gonzalez, USACE Risk Management Center, julia.nosp@m.n.t..nosp@m.gonza.nosp@m.lez@.nosp@m.usace.nosp@m..arm.nosp@m.y.mil

Description:

This class contains a shared function for finding the solution to the equation, where f(x)=0 using the Secant method. The secant method is a root-finding procedure in numerical analysis that uses a series of roots of secant lines to better approximate a root of a function.

References:

  • "Numerical Recipes, Routines and Examples in Basic", J.C. Sprott, Cambridge University Press, 1991.

Use the secant method to find a solution of the equation where f(x)=0.

Parameters
fThe function to solve.
lowerBoundThe lower bound (a) of the interval containing the root.
upperBoundThe upper bound (b) of the interval containing the root.
toleranceOptional. Desired tolerance for both the root and the function value at the root. The root will be refined until the tolerance is achieved or the maximum number of iterations is reached. Default = 1e-8.
maxIterationsOptional. Maximum number of iterations. Default = 1000.
reportFailureOptional. If set to true, an exception will be thrown if the routine fails to converge. If set to false, the root from the last iteration will be returned if the routine fails to converge. Default = True.
Returns
The root to the equation f(x)=0 given the specified tolerance.
