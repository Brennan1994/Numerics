Detailed Description

The k-Nearest Neighbors (k-NN) algorithm.

Authors: Haden Smith, USACE Risk Management Center, cole..nosp@m.h.sm.nosp@m.ith@u.nosp@m.sace.nosp@m..army.nosp@m..mil

Description:

k-NN is a simple method for applying supervised learning. When training a k-NN model, the model practically memorizes all locations of all the points and their values. When it is time to use the model for predictions, the model takes the input data and calculates the k-nearest points to that input.
