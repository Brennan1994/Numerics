Detailed Description

Contains Fourier transform methods.

Authors: Haden Smith, USACE Risk Management Center, cole..nosp@m.h.sm.nosp@m.ith@u.nosp@m.sace.nosp@m..army.nosp@m..mil

Description: This class contains shared functions for Fast Fourier Transform (FFT), an algorithm that computes the Discrete Fourier Transform (DFT) which is obtained by decomposing a sequence of values into components of different frequencies. The DFT converts a finite sequence of equally-spaced samples of the discrete-time Fourier transform (DTFT), a complex valued function of frequency. A FFT rapidly computes these transformations by factorizing the DFT matrix into a product of mostly zero factors.

References:

  • "Numerical Recipes, Routines and Examples in Basic", J.C. Sprott, Cambridge University Press, 1991.

Calculates the Fourier transform of a set of n real-valued data points.

Parameters
dataData is a real array of length n. n must be an integer power of 2.
inverseDetermines whether to perform an inverse transform.

Replaces these data by the positive frequency half of their complex Fourier transform. The real-valued first and last components of the complex transform are returned as elements data(0) and data(1), respectively. This routine also calculates the inverse transform of a complex data array if it is the transform of real data. (Result in this case must be multiplied by 2/n.)
