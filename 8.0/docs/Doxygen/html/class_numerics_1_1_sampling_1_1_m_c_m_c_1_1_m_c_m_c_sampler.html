Detailed Description

A base class for all Markov Chain Monte Carlo (MCMC) samplers.

Authors: Haden Smith, USACE Risk Management Center, cole..nosp@m.h.sm.nosp@m.ith@u.nosp@m.sace.nosp@m..army.nosp@m..mil

◆ InitializationType

enum Numerics.Sampling.MCMC.MCMCSampler.InitializationType

Enumerates the initialization types.

Enumerator
MAP 

Initialize the chains using the Maximum a Posteriori (MAP) estimate and covariance matrix. If the MAP optimization fails, chains will be automatically initialization with random samples from the priors.

Randomize 

Automatically initialize the chains with random samples from the priors. This is the default.

UserDefined 

Initialize the chains from user-defined points.
