Detailed Description

Class designed to store x-y data that is ordered for both the x and y values.

Authors: Woodrow Fields, USACE Risk Management Center, woodr.nosp@m.ow.l.nosp@m..fiel.nosp@m.ds@u.nosp@m.sace..nosp@m.army.nosp@m..mil

Description: The most common use case for this class is to store curve data such as a cumulative distribution function. One of the more powerful functions associated with this class is the Transform() function which allows to transform the OrderedPairedData class with another to create a new class. Common use cases of the Transform() function include transforming a flood frequency function into a stage frequency function using a flow-stage rating curve.
