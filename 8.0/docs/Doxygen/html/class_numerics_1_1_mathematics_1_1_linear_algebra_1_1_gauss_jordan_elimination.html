Detailed Description

A class for solving a set of linear equations Gauss-Jordan elimination.

Authors:

  • Haden Smith, USACE Risk Management Center, cole..nosp@m.h.sm.nosp@m.ith@u.nosp@m.sace.nosp@m..army.nosp@m..mil
  • Tiki Gonzalez, USACE Risk Management Center, julia.nosp@m.n.t..nosp@m.gonza.nosp@m.lez@.nosp@m.usace.nosp@m..arm.nosp@m.y.mil

Description:

Gauss-Jordan Elimination is a row reduction algorithm that correctly formats matrices so that we can solve linear systems in the form A*x = b.
