Detailed Description

Class for performing Box-Cox transformation.

Authors: Haden Smith, USACE Risk Management Center, cole..nosp@m.h.sm.nosp@m.ith@u.nosp@m.sace.nosp@m..army.nosp@m..mil

Description: This method transforms non-normal dependent variables into a normal shape.
