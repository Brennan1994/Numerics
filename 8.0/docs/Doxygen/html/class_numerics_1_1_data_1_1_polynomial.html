Detailed Description

A class for polynomial interpolation.

Authors: Haden Smith, USACE Risk Management Center, cole..nosp@m.h.sm.nosp@m.ith@u.nosp@m.sace.nosp@m..army.nosp@m..mil

Description: This method is the interpolation of a given bivariate data set by the polynomial of lowest possible degree that passes through the points of the dataset.

References:

  • "Numerical Recipes, Routines and Examples in Basic", J.C. Sprott, Cambridge University Press, 1991.
