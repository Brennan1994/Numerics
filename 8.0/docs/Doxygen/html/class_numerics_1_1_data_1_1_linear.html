Detailed Description

A class for linear interpolation.

Authors: Haden Smith, USACE Risk Management Center, cole..nosp@m.h.sm.nosp@m.ith@u.nosp@m.sace.nosp@m..army.nosp@m..mil

Description: Linear interpolation is a method of estimation and curve fitting using linear polynomials to construct new data points within the range of a discrete set of known data points.

Methods are designed to be compatible with the VBA Macro 'Interpolate Version 2.0.0' November 2017, USACE Risk Management Center
